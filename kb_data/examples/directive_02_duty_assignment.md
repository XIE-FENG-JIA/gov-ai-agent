# 手令

### 主旨
茲令經濟發展局副局長周志遠先生兼任「亞洲新灣區2.0開發計畫專案辦公室」主任，統籌推動相關業務，希即遵照辦理。

### 指示事項
一、周志遠先生於兼任專案辦公室主任期間，仍應兼顧經濟發展局副局長本職業務。
二、專案辦公室設於經濟發展局八樓第二會議室，自115年3月1日起正式運作。
三、專案辦公室編制人員由都市發展局、工務局、交通局、海洋局及經濟發展局各調派一至二名專責人員，受專案辦公室主任指揮調度。
四、專案辦公室每二週應向本人提出工作進度報告，遇重大事項應即時報告。

### 說明
一、依據高雄市政府組織自治條例第二十條及市政會議114年12月27日第45次會議決議辦理。
二、亞洲新灣區2.0開發計畫為本市115至119年度重大施政計畫，總投資額預估新臺幣1,800億元，涵蓋智慧科技園區、國際會展中心擴建、港區文創聚落及環港輕軌延伸等四大子計畫，需跨局處整合推動。
三、周志遠先生具都市計畫及產業發展雙重專業背景，曾擔任前鎮加工出口區轉型計畫協調人，熟悉新灣區開發事務。

### 完成期限
一、115年3月1日前完成專案辦公室場地及人員到位。
二、115年3月31日前提出亞洲新灣區2.0開發計畫整體執行方案。

### 副知
高雄市政府人事處、都市發展局、工務局、交通局、海洋局、經濟發展局

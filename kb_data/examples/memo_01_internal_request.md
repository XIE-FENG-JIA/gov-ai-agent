# 箋函

### 主旨
本處因辦公空間擴增新增同仁進駐，亟需資訊設備一批，請 查照惠予撥借。

### 說明
一、本處為因應市政信箱業務量成長，經簽奉核准自115年3月1日起擴增市政信箱管理科編制員額5人，新進同仁將進駐市府大樓北區10樓1005室。
二、所需資訊設備規格及數量如下：
　（一）桌上型電腦主機5台（含螢幕），規格需求：處理器Intel Core i5以上、記憶體16GB以上、固態硬碟512GB以上、23吋螢幕。
　（二）多功能事務機1台：需具備列印、影印、掃描及傳真功能，每分鐘列印速度30頁以上。
　（三）網路交換器1台：24埠Gigabit乙太網路交換器。
　（四）IP電話機5台：需支援市府內部VoIP系統。
三、前述設備如貴局現有庫存不足撥借，請惠予告知不足品項，本處將另行辦理採購。
四、請於115年2月28日前回覆可撥借之設備項目及預定交付時間。
